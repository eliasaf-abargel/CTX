# CTX Provider Logos

Optional provider logo files can be placed here so `ProviderIcon` can render
official marks in the app.

Expected filenames:

| Provider | File |
|---|---|
| AWS | `aws.png` |
| GCP | `gcp.png` |
| Azure | `azure.png` |
| Kubernetes | `kubernetes.png` |

Guidelines:

- Use transparent PNG files when possible.
- Keep each vendor mark unmodified and follow the vendor's brand guidelines.
- If a file is missing, CTX falls back to a monochrome SF Symbol so the app
  still builds and runs.
